PHP

  1. What makes PHP important for creating dynamic websites? Discuss the primary benefits of PHP compared to other server-side scripting languages.
  2. Compare the roles of PHP and HTML in web development. How do their purposes and functionalities differ?
  3. How does PHP process form data using the GET and POST methods? Explain the potential security vulnerabilities tied to each method.
  4. Develop a PHP program that showcases the application of arithmetic, relational, and logical operators.
  5. Outline the various looping structures in PHP. Provide a PHP script demonstrating the use of while, for, and do-while loops.
  6. What does a function mean in PHP? How is it defined and called? Create a basic PHP function that adds two numbers and returns the sum. Additionally, define a recursive function in PHP and provide an example that computes the factorial of a number.
  7. What is a session in PHP, and why is it critical for web development? Explain how sessions enable the persistence of user data across multiple pages.
  8. How do PHP sessions operate? Write a PHP program that starts a session, stores a user’s name in a session variable, and retrieves it on a different page.
  9. What are the advantages and limitations of using cookies for session management? Describe situations where cookies are an appropriate choice.
  10. How can you set a cookie in PHP? Create a PHP script that stores a user’s name in a cookie and displays it on another page.
  11. Why is robust error and exception handling vital in PHP? How does it enhance an application’s security and dependability?
  12. Distinguish between PHP errors and exceptions. Include examples of each and describe their respective handling methods.
  13. What role does the throw keyword play in PHP exception handling? How does it differ from the catch block?
  14. How can you activate error reporting in PHP for debugging? Write a PHP script using error_reporting() and ini_set() to show all errors during development.
  15. Why is database integration essential in PHP-based web applications? How does it support the creation of dynamic content?
  16. What approaches exist to link PHP with a MySQL database? Evaluate mysqli and PDO based on security, usability, and functionality.
  17. Provide SQL statements to create a MySQL database and a table called users with columns like id, name, and email.
  18. Write a PHP script using mysqli to connect to a MySQL database. Incorporate error handling to indicate whether the connection succeeded or failed.
